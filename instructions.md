## Step 1: Navigate to play.blooket.com
First step is to get your game ready to play to use the hacks. In your browser navigate to play.blooket.com.
You are going to join a game or host a game of blooket on your own.
Once the game is setup and you have joined proceed to the next step.

## Step 2: Get the Loader script
Well ya kinda need some way to run the script. In the main GitHub repository look for the **"run-script.md"** file and open it
You will see a bunch if text at the top and them a javascript: script at the bottom.
Under "PASTE THIS" copy the javascript: script. MAKE SURE YOU COPY THE JAVASCRIPT: PART!

## Step 3: Run the script
Now that you have copied the script from the GitHub you need to run the script.
Navigate back to your play.blooket.com tab and go to the search bar. Delete the play.blooket.com URL in there and paste the script you copied.
Once that is pasted press "Enter". A notification from your browser should pop up. It should say something along the lines of "play.blooket.com says: If the cheats don't run. Check if the Github is down."
Press OK and the K-GUI should pop up. Don't see anything? Check below.

### Common Issues
1. You forgot the Javascript: in front of the script and it ended up as a google search.
2. You didn't copy the entire script. Make sure you are copying entire script even one missing letter could make the script not work.
3. You copied too much of the file. Make sure you don't copy the comments on the top of the run-script.md file. Only copy what is below the "PASTE THIS" text.
4. The GitHub Repo is down wait for it to come back up.
5. If you are still having issues, make a bug report in the issues tab of the repo.
